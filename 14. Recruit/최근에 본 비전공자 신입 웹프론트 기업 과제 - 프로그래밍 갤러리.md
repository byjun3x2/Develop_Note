## 갤러리 본문 영역

  

여러 곳 지원했다가 3곳에서 연락 왔는데 3곳 다 과제를 주 갰다고 함  

  

**1번 회사**: PDF 주면서 30분 내로 해결하라고 해서 열어보니 문제가 2개 있고 하나는 계산기 이미지 있고 HTML과 CSS로 똑같이 구현하라고 함, 나머지 하나는 구구단 을 console.log로 2345, 6789를 끊어서 출력되게 하라고 함 계산기는 이미지만 딸랑 하나 있어서 생상 코드 padding 등 스타일 값 없이 눈대중으로 만들라는 뜻인데 CSS랑 HTML만 사용하는 건 정말 오랜만이라 이미지랑 만들고 있는 걸 서로 비교해가면서 짜느라 25분 정도 걸렸고 나머지 구구단은 한 4분 만에 짜긴 했는데 지금 생각해 보니 오타가 있어서 망했음

**사용 기술**: HTML, CSS, JavaScript

  

  

**2번 회사** : 2번 회사는 실무자 면접 > 과제 > 임원 면접인데 실무자 면접 때 서버는 아직 잘 모른다고 했더니 과제를 4일 동안 다른 사이트 메인 페이지를 클론 코딩하고 메인 페이지의 데이터 API를 직접 만들어 AWS EC2에 서버를 배포한 다음 API로 받아와서 출력하라고 함, 그래서 서버를 처음부터 공부하다가 좀 더 퍼포먼스 있게 하려고 DB까지 공부하고 AWS EC2에 DB도 같이 배포한 다음 CRUD를 구현했는데 시간이 부족해서 프론 트를 제대로 못 만들어서 떨어짐

**사용 기술** : Next.js, TypeScript, Express, MongoDB, AWS EC2......

  

  

  

**3번 회사**: 여기는 1주일 동안 간단한 UI에 탭을 추가해 탭을 이동하는 기능과 Web3를 사용해서 메타 마스크? 로그인을 구현하고 로그인 데이터로 이더리움의 지갑 잔액과 지갑 주소를 가져와 출력하라고 함 가상화폐에는 관심이 없어서 메타 마스크가 뭔지도 몰랐는데 일단 개발 시작했는데 로그인 기능은 금방 만들었고 시간이 남아서 반응형으로 만들고 퀄리티가 마음에 안 들어서 무한 스크롤도 구현하고, Redux로 상태 관리도 하고, 메타 마스크를 지원하는 브라우저 체크 기능도 추가하고 netlify에 배포하고 사이트 링크와 깃허브 링크를 보내줬는데 반응이 좋아서 최종 면접 오라고 연락 옴

**사용 기술**: React.js, Redux, TypeScript, Web3, tailwindcss......
